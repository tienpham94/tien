My first NPM library for react  styled components.

Getting started
---------------

Install `tien` using npm.

```shell
npm install tien --save
```

Example usage:

```js
import React from 'react'
import { H1, H2, H3, H4, Button } from 'tien'

const App = () =>
  <div>
    <H1>Heading 1</H1>
    <H2>Heading 2</H2>
    <H3>Heading 3</H3>
    <H4>Heading 4</H4>
    <Button type="submit">Hello world</Button>
  </div>

export default App
```
